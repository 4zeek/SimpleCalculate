# SimpleCalculate
Простой калькулятор на питоне 

# Работа
Просто вводите 2 числа,какое действие выполнить и все!

## Примеры
- Целые числа
```text
1 Number ->2
2 Number ->2
action (+,-,*,/) ->+
Result -> 4.0
```
- Дробные числа
```text
1 Number ->2.2
2 Number ->3.4
action (+,-,*,/) ->+
Result -> 5.6
```
